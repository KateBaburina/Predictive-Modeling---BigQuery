## Name
Predictive Modeling - BigQuery

## Description
The dataset from Kaggle.com was used for creating this predictive model https://www.kaggle.com/tejashvi14/employee-future-prediction. A company's HR department wants to predict whether some employees would leave the company in next 2 years. The task is to build a predictive model that predicts the prospects of future and present employee. Google BigQuery was used for the model creation.

## Task

The task is to build a model to solve a binary classification task on unbalanced data. The PDF file is the report created using the data in Google BigQuery.

## Algorithms used

Google BigQuery - Linear Regression, XGB

## Metrics used

ROC AUC, Log Loss
